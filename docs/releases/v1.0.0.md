## BTCC Wallet v1.0.0

首个 **All-in-One** macOS 客户端：BTCC 助记词钱包 + 矿池算力 + Apple Silicon GPU 挖矿。

### 下载

- **BTCC-Wallet-v1.0.0.dmg** — 拖到「应用程序」即可安装
- 需要 Xcode Command Line Tools（`xcode-select --install`）

### 新功能

- **BIP39 钱包**：创建/导入助记词、cc1 地址、余额查询、转账
- **矿池算力**：pool.btc-classic.org API 查询矿工算力与待结算
- **GPU 挖矿**：Stratum 矿池挖矿、SOCKS5/HTTP 代理、M 系列零调参 ~180 MH/s
- **SwiftUI 原生界面**：钱包优先 Tab 布局，一键钱包地址用于挖矿

### 整合说明

由 [BTCC_apple-gpu-miner](https://github.com/wendell1224/BTCC_apple-gpu-miner) GUI 项目整合迁移，应用重命名为 **BTCC Wallet**。
