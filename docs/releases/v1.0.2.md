# BTCC Wallet v1.0.2

## 新增

- 新增 Electron + Node.js 桌面客户端，SwiftUI 版本保留在 `gui/native/`。
- 钱包助记词改为用户密码加密存储，使用 `scrypt` + `AES-256-GCM`，不依赖 macOS Keychain。
- 新增矿池 / Solo 低功耗在线模式，可设置 5%-100% GPU 平均占用。
- 矿池算力页面新增 Solo Top Hashrates 排行榜，展示完整地址、Worker、1h/1d/7d 算力和 Best Share。

## 修复与优化

- 支持 Native SegWit、Taproot、Legacy、P2SH 地址互转。
- 转账成功后展示 TXID 和区块浏览器链接。
- 钱包历史改用区块浏览器分页接口，展示转账、收到、自转账等动作。
- OTC 页面展示最新价格、24h 涨跌、成交量、成交额和累计数据。

## 验证

- Node 钱包 / 交易 / 矿工核心单测通过。
- Python 核心回归测试通过。
- Electron 冒烟测试通过。
- SwiftUI 旧版类型检查通过。
